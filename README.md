### Prerequisites

Use nvm install Node.js 
Ref: [GitHub](https://github.com/creationix/nvm)

```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
```
```
command -v nvm
```

### Installing

Install in the folder

```
npm i express
npm i express-handlebars
npm i body-parser
```
